# Contributing to BearClawWeb

## Setup

```bash
bundle install
bundle exec rails test
```

## Expectations

- Keep service integration behavior explicit and well documented.
- Update `README.md` when routes, configuration, or operator-visible behavior changes.
- Update `BLINK.md` when the build, deploy, or verification contract changes.
- Update `SECURITY.md` for auth, token, or secret-handling changes.
- Update `CHANGELOG.md` for notable repo changes.
